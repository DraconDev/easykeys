# easykeys README

## Working with Markdown

A very simple intuitive hotkey pack to help with keyboard navigation.

alt e: Row up
alt d: Row down

alt s: Left by one word
alt f: Right by one word

alt w: Start of line
alt r: End of line

alt x: Left by one char
alt v: Right by one char

alt 3: Top of page
alt c: Bottom of page
