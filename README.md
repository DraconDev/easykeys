# Easykeys

A very simple intuitive hotkey pack to help with keyboard navigation.

-alt w: Row up
-alt s: Row down

-alt a: Left by one word
-alt d: Right by one word

-alt q: Start of line
-alt e: End of line

-alt z: Left by one char
-alt c: Right by one char

-alt 2: Top of page
-alt x: Bottom of page
